# 🌾 KRISHIRAKSHAK-AI

### AI Powered Smart Farming Assistant

KrishiRakshak-AI is an **AI-powered agriculture assistant platform**
designed to help farmers detect plant diseases, understand crop health,
and receive intelligent treatment suggestions.

The system combines **Artificial Intelligence, image analysis, and
modern web technologies** to provide farmers with instant plant health
diagnostics and actionable insights.

------------------------------------------------------------------------

# 🚜 Project Vision

Agriculture is one of the most important sectors of the economy.
However, many farmers struggle to quickly identify crop diseases.

**KrishiRakshak-AI aims to solve this problem by providing:**

-   Instant plant disease detection
-   AI generated treatment recommendations
-   Agricultural knowledge resources
-   Digital crop diagnosis history
-   Smart farming assistance

This system helps farmers **reduce crop loss, increase productivity, and
make better farming decisions.**

------------------------------------------------------------------------

# ✨ Core Features

## 🌿 Plant Disease Detection

Farmers can upload images of plant leaves and the system analyzes them
to detect possible diseases.

## 🤖 AI Generated Recommendations

The AI engine provides: - Disease explanation - Treatment suggestions -
Prevention tips

## 📚 Agricultural Knowledge Library

Database containing information about common plant diseases and farming
guidance.

## 🗂 Plant Diagnosis History

Stores previous plant analysis results using **SQLite database**.

## 👨‍🌾 Farmer Support System

Provides guidance resources and contact information for agricultural
support.

# ⚙️ Installation Guide

## 1. Clone the Repository

git clone https://github.com/Ameershaik3288/KRISHIRAKSHAK-AI

cd KRISHIRAKSHAK-AI

## 2. Install Dependencies

npm install

## 3. Configure Environment Variables

Create a .env file and add:

GOOGLE_API_KEY=your_api_key_here

## 4. Run Development Server

npm run dev

## 5. Build Project

npm run build
